## Projects

**RAFT-Stereo Optimization and Lightweight Deployment**  
Optimized RAFT-Stereo for binocular stereo matching, focusing on refinement strategies, edge-aware loss, partial parameter unfreezing, and FP16 inference acceleration.

- Built experiments on the Middlebury2014 stereo dataset.
- Compared baseline, refinement-only training, refinement with edge-aware loss, and partial unfreezing.
- Reduced model size from about 43 MB to 22 MB with FP16 inference and improved inference speed from about 209.80 ms to 173.59 ms per image pair while maintaining comparable accuracy.

**Competitive Price Monitoring System for E-commerce**  
Python and Playwright-based automation system for SKU-level price monitoring across Taobao/Tmall stores.

- Designed browser automation workflows for collecting product information and SKU-level pricing.
- Explored LLM-based SKU normalization and product model matching.
