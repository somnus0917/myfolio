This project designs an automation workflow for collecting Douyin Shop Compass analytics data across multiple stores.

## Highlights

- Designed a workflow based on manual login, persistent browser sessions, store-level profile switching, report downloading, and local parsing.
- Planned a local data pipeline from exported Excel/CSV files to a dashboard.
- Considered practical risk control issues in browser automation.

## Context

The goal is to reduce repetitive reporting work while keeping the workflow realistic for systems that require browser sessions, manual authentication, and store-level context switching.
