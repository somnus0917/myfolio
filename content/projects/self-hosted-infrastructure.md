This project collects self-hosted services and personal web systems deployed around the `somnus.top` domain.

## Highlights

- Built and maintained personal websites and documentation systems using Hugo, Quartz, Ghost, and GitHub Pages.
- Deployed self-hosted services such as Vaultwarden, AFFiNE, Uptime Kuma, and Caddy reverse proxy.
- Explored Cloudflare, DNS, R2-based image hosting, and custom subdomain routing.
- Domains include `somnus.top`, `docs.somnus.top`, `paper.somnus.top`, `portfolio.somnus.top`, and `status.somnus.top`.

## Context

The infrastructure is both a personal productivity system and a long-running engineering playground for deployment, observability, documentation, and self-hosted workflows.
